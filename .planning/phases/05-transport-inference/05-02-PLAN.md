Create a new test file `tests/testthat/test-inference-calibration.R` with tests covering:

1. **Null distribution uniformity test** (core requirement):
   - Generate pure noise data: S=20 subjects, Q=10 clusters, values ~ N(0,1)
   - Run `dkge_signflip_maxT(Y, B=500)` per CONTEXT.md decision
   - Collect p-values across multiple iterations (e.g., 100 independent noise datasets)
   - Bin p-values into 10 bins (0-0.1, 0.1-0.2, ..., 0.9-1.0) per RESEARCH.md
   - Apply chi-square test: `chisq.test(observed_counts, p=rep(0.1, 10))`
   - Test should pass at alpha=0.05 (p-value > 0.05 indicates uniform distribution)

2. **Shuffled labels null construction**:
   - Create structured data with true signal (effect in some clusters)
   - Shuffle labels to break true association (permute Y rows independently per cluster)
   - Verify p-values from shuffled data are uniform (chi-square passes)
   - This mirrors standard neuroimaging permutation practice per CONTEXT.md

3. **Single-cluster sanity check**:
   - Create data where one cluster has strong effect (all subjects positive)
   - Verify that cluster gets small p-value (< 0.05)
   - Other clusters (noise) should have uniform p-values

Use set.seed(42) at start for reproducibility, but note chi-square result depends on random draws.
