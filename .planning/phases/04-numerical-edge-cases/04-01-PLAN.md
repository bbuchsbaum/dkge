  <name>Task 1: Add numerical robustness utility functions</name>
  <files>R/dkge-utils.R</files>
  <action>
Add three internal helper functions to R/dkge-utils.R:

1. `.dkge_check_rank(design, beta = NULL, subject_id = NULL)`:
   - Check design matrix rank via `qr(design)$rank`
   - If rank < ncol(design), emit warning with subject_id and rank info
   - Optionally check beta matrix rank similarly
   - Return list(design_rank, beta_rank) for metadata tracking

2. `.dkge_check_condition(M, threshold = 1e8, name = "matrix")`:
   - Compute condition number via `kappa(M, exact = FALSE)`
   - If condition > threshold, emit warning with condition number and threshold
   - Return the condition number for metadata

3. `.dkge_voxel_exclusion_mask(B_list, subject_ids = NULL)`:
   - For each beta matrix, identify columns with any non-finite values
   - If exclusions found, emit warning per subject with count and percentage
   - Return list(excluded_voxels, excluded_counts, total_excluded) for metadata

Use warning(..., call. = FALSE) for clean messages. Use %||% operator from rlang if available, otherwise define locally.

Follow existing codebase style: roxygen2 @keywords internal, @noRd for non-exported functions.
