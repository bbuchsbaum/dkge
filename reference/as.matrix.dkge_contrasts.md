# Extract contrast values as matrix

Extract contrast values as matrix

## Usage

``` r
# S3 method for class 'dkge_contrasts'
as.matrix(x, contrast = 1, ...)
```

## Arguments

- x:

  A dkge_contrasts object

- contrast:

  Name or index of contrast to extract

- ...:

  Additional arguments (not used)

## Value

SxP matrix of contrast values
