# Principal-angle cosines in the K-metric

Principal-angle cosines in the K-metric

## Usage

``` r
dkge_cosines_K(U, V, K)
```

## Arguments

- U:

  Matrix with \`q\` rows.

- V:

  Matrix with \`q\` rows.

- K:

  Symmetric positive (semi-)definite matrix defining the metric.

## Value

Numeric vector of singular values in \[0, 1\].
